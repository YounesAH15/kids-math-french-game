**LudiKids** is a world-class, premium educational platform designed for toddlers and early childhood development. It combines a massive, curated vocabulary database with tactile interactivity and "juicy" animations to create a stunning learning environment.

> [!NOTE]
> **Total Content Milestone**: **100 Categories | ~800 Bilingual Items (FR/EN)**.
> This project has been scaled to an encyclopedic level, covering everything from daily objects to Ancient Greece and Deep Space.

---

## ✨ Key Features

- **🛡️ The Century 100 Content Library**: 100 categories covering Animals, Science, History, Hobbies, and more.
- **🌐 100% Bilingual Mastery**: Toggle instantly between **French** and **English**. Every single item has unique localized names and voice support.
- **🎙️ AI Voice Synthesis**: Integrated Web Speech API provides cute, kid-friendly auditory feedback for every word.
- **🕹️ Interactive Playground**: A free-form, tactile environment where children can play with their unlocked stickers. Drag-and-drop state is saved automatically.
- **📖 Sticker Book Collection**: A gamified reward system where children earn stars in Math and Vocabulary games to unlock rare stickers.
- **🧠 Adaptive Math Engine**: Dynamic math challenges covering counting and base-10 building (Tens/Units).

## 🎨 Design Aesthetics & UX

- **Glassmorphism Layering**: A modern, translucent UI that feels premium and alive.
- **Squash & Stretch Animations**: "Juicy" interactions that provide tactile satisfaction for small hands.
- **Smooth Scene Transitions**: Animated slide transitions between the Intro, Games, Sticker Book, and Playground.

## 🛠️ Technology Stack

- **Core**: Vanilla JavaScript (ES6+), HTML5 Semantic Structure.
- **Styling**: Modern CSS3 (Variables, Flexbox, Grid, Glassmorphism).
- **Audio**: Web Speech API & Web Audio API (Oscillator synth).
- **Persistence**: LocalStorage for stars, unlocked stickers, and playground positions.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/kids-math-french-game.git
   ```
2. **Install dependencies** (using Vite):
   ```bash
   npm install
   ```
3. **Run the dev server**:
   ```bash
   npm run dev
   ```
